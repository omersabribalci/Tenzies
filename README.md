# Tenzies Game (TypeScript Practice)

This project is a small implementation of the Tenzies dice game, built for practicing **React with TypeScript**.

## Features
- Roll and hold dice until all match
- Move counter
- Best record tracking (moves)
- Fastest game tracking (time)
- Records saved with LocalStorage

## Purpose
Created as a practice project to explore React state management, TypeScript typing, and simple persistence with LocalStorage.
